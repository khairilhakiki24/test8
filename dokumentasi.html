Dokumentasi SketchPro

Panduan Lengkap Penggunaan & Instalasi (Versi 4.0)

1. Panduan Penggunaan Fitur Utama

  • Menggambar Presisi (Bezier Smoothing): Gunakan alat Pena untuk menggambar. Sistem secara otomatis akan menghaluskan goresan tangan Anda menggunakan algoritma Quadratic Bezier Curve.
  • Sistem Multi-Layer Profesional: Anda dapat menambahkan Teks, Bentuk Geometri (Shapes), dan Gambar ke dalam kanvas. Setiap objek beroperasi dalam layernya masing-masing. Buka panel Layers di kanan atas untuk mengunci (Lock), menyembunyikan (Hide), atau mengubah urutan objek (Z-Index).
  • Smart Guides & Multi-Select: Saat menggeser objek, garis bantu merah (Smart Guides) akan otomatis muncul untuk membantu Anda menyejajarkan objek dengan presisi. Tahan tombol Shift atau Ctrl lalu klik beberapa objek untuk menyeleksi banyak layer sekaligus, atau tekan Ctrl + A untuk memilih semua.
  • Grup & Pintasan (Shortcuts): Klik kanan pada kanvas untuk membuka Context Menu. Anda bisa melakukan duplikasi, memindah posisi layer (depan/belakang), serta menggabungkan layer (Group/Ungroup).
  • Ekspor High-Res (300 DPI): Klik tombol Ekspor (ikon Unduh Awan). Anda bisa merender kanvas hingga resolusi tinggi untuk dicetak ke format PDF, JPG, atau PNG dengan dukungan latar belakang transparan.
  • Auto-Save (Zero Lag): Sistem Delta History akan terus menyimpan progres Anda di latar belakang. Jika browser tidak sengaja di-refresh, kanvas Anda tidak akan hilang.

2. Panduan Personalisasi Brand (Warna & Nama)

Aplikasi ini dirancang untuk mudah di-branding ulang sesuai keinginan Anda atau klien:

  1. Buka Menu Pengaturan di navigasi bawah/samping.
  2. Di kolom Nama Brand Aplikasi, ketikkan nama aplikasi yang Anda inginkan.
  3. Di kolom Warna Tema Utama, pilih warna melalui Color Picker (Warna ini akan otomatis mengubah seluruh tombol, ikon aktif, garis batas seleksi, dan status bar HP Android).
  4. Klik tombol Simpan.

3. Panduan Modifikasi Icon PWA

Untuk membuat aplikasi terinstal sempurna layaknya aplikasi asli (Native) di Android/iOS, siapkan icon logo Anda:

  • Buat folder baru bernama assets satu direktori dengan file index.html.
  • Siapkan 2 buah file gambar berformat PNG.
  • Beri nama file pertama: icon-192x192.png (Ukuran tepat 192x192 pixel).
  • Beri nama file kedua: icon-512x512.png (Ukuran tepat 512x512 pixel).
  • Letakkan kedua file tersebut di dalam folder assets/.

4. Panduan Memecah File (Untuk Developer)

Bagi Anda yang ingin merapikan struktur file ke format folder web standar (CSS/JS terpisah), buka index.html dan cari komentar pembatas berikut:

<!-- ============================================== -->
<!-- FILE: source/assets/css/style.css START -->
... pindahkan seluruh kode CSS di sini ke file style.css ...
<!-- FILE: source/assets/css/style.css END -->

Lakukan hal yang sama pada blok kode JavaScript di bagian paling bawah file (cari komentar pemisah app.js).

Catatan Mode Offline: Aplikasi ini dilindungi oleh Service Worker. Namun, untuk penggunaan 100% Offline (tanpa koneksi internet sama sekali saat instalasi pertama), Anda wajib mengunduh link CDN (Tailwind, Lucide, SheetJS, jsPDF, ChartJS) yang ada di tag <head> dan mengubah jalurnya (path) ke file lokal Anda.

5. Backup & Restore Data Database

Aplikasi ini berjalan secara lokal dan private di perangkat Anda (menggunakan IndexedDB). Oleh karena itu:

  • Gunakan fitur Backup Data di Menu Pengaturan secara berkala untuk mengekspor data metadata Galeri dan Kategori ke dalam format Excel (.xlsx).
  • Jika Anda berganti perangkat atau browser, gunakan fitur Restore Data dan unggah file backup Excel Anda sebelumnya untuk mengembalikan struktur galeri.
